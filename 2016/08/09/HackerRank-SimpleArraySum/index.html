HackerRank - Simple Array Sum

Aug 9, 2016

這個題目呢~
標準輸入兩次
1.) 數組 的個數 N
2.) 用空白間隔的 N 整數數組

要輸出數組的和.

Input Format
The first line contains an integer, NN N, denoting the size of the array. The second line contains NN N space-separated integers representing the array’s elements.

Output Format
Print the sum of the array’s elements as a single integer.
