install pyodbc on macOS Sierra

Sep 12, 2017

由於新版的 os x(macOS) 已經將 sql.h 拿掉了
所以直接 pip install pyodbc 時
c++ compile 會找不到 header 檔
選項就是安裝 iODBC 或是裝 unixODBC
但我不想裝 iODBC, 所以選了 unixODBC
拜新版 OS X 的 SIP 機制所賜~

整體來說裝 pyodbc 變的難度高上許多

簡單記錄一下後來怎麼裝成的

大概是這樣.😓
